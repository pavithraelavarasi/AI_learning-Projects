HTML Tags
Tags are the building blocks of every web page. They tell the browser what kind of content to show.
What is a tag?

An HTML tag is a keyword in angle brackets: <tagname>. Most come in pairs — an opening and closing tag. The content sits between them. Tags don't make things look good — they give content meaning.

HTML
<h1>Your Name</h1> ← big heading <p>Short bio note here.</p> ← paragraph <img src="photo.jpg"> ← self-closing, no / <div> <p>I wrap other tags.</p> ← nesting </div>

Lesson 02 / 04
HTML Images
The <img> tag displays your photo or avatar. It's self-closing and controlled via attributes.
The img tag

<img> has no closing tag — it's self-closing. You describe the image through attributes. The two essential ones are src (where the image is) and alt (a text description for accessibility).

HTML
/* Minimum required */ <img src="photo.jpg" alt="My photo"> /* With size */ <img src="photo.jpg" alt="My photo" width="80" height="80"> /* Circular avatar with CSS */ <img src="photo.jpg" alt="Avatar" style="width:80px; height:80px; border-radius:50%; object-fit:cover">
🔗
src
URL or file path. Can be a link from the web too.
🏷️
alt
Text shown if image fails. Always include it.
📐
width / height
Size in pixels. Or use CSS for more control.
⭕
border-radius: 50%
Makes any image a perfect circle in CSS.

Lesson 03 / 04
CSS Padding
Padding is the space inside an element — between the content and the element's edges.
The cushion inside

Imagine a profile card with no padding. The text would be jammed against the edges. Padding is the breathing room inside the box. The element's background colour fills the padding area too.

CSS
.card { padding: 24px; } /* all 4 sides */ .card { padding: 16px 24px; } /* top/bottom left/right */ .card { padding-top: 20px; padding-right: 24px; padding-bottom: 20px; padding-left: 24px; } /* individual */
Try it — adjust the padding
▶ Try It Change the padding value and watch the card breathe
💡 Try padding: 4px vs padding: 40px. Notice how larger padding makes the card feel more open and premium.
Lesson 04 / 04
CSS Margin
Margin is the space outside an element — the gap between it and its neighbours.
Space outside the box

If padding is the cushion inside the box, margin is the empty space around the outside. It pushes other elements away. The background colour does NOT fill margin — it's completely transparent space.

MARGIN — outside space
BORDER
PADDING — inside space
Content
CSS
.card { margin: 32px; } /* all sides */ .card { margin: 0 auto; } /* centre horizontally ← very useful! */ .name { margin-bottom: 8px; } /* space below the name */ .bio { margin-top: 12px; } /* space above bio */
Try it — space between elements
▶ Try It Change margin values — try 0, 8px, 32px
🎯 The rule: padding = space inside the box. margin = space outside between boxes.
