What is Flexbox?
Flexbox is CSS's one-dimensional layout system — it makes arranging items in a row or column effortless.
The big idea

Before Flexbox, centering things in CSS was famously painful. Flexbox changed that. It introduces a flex container and flex items inside it. You tell the container how to distribute space and align its children — and it just works.

You activate it with a single line: display: flex. The container becomes flex, and all its direct children become flex items automatically.

CSS
/* Step 1: Make the parent a flex container */ .container { display: flex; /* ✨ That's it! */ } /* All direct children become flex items automatically */ .item { /* No special rules needed */ }
Main axis vs Cross axis

Flexbox works along two axes. The main axis is where items are laid out (horizontal by default). The cross axis is perpendicular to it.

Lesson 02 of 05
justify-content
Control how items are distributed along the main axis — the direction your flex items flow.
What it does

justify-content decides what happens to the leftover space on the main axis. You can push all items to one end, spread them out evenly, or put space between them — without touching the items themselves.

CSS
.container { display: flex; justify-content: flex-start; /* default */ justify-content: flex-end; /* push to end */ justify-content: center; /* centre them */ justify-content: space-between; /* max gap between */ justify-content: space-around; /* equal space around */ justify-content: space-evenly; /* identical gaps */ }

Lesson 03 of 05
align-items
Control how items sit on the cross axis — the axis perpendicular to your main direction.
Cross axis alignment

While justify-content handles the main axis, align-items handles the cross axis. By default it's stretch — items expand to fill the container's height. Change it to center and everything magically centres vertically.

CSS
.container { display: flex; align-items: stretch; /* default — fill height */ align-items: flex-start; /* top of container */ align-items: flex-end; /* bottom of container */ align-items: center; /* perfect vertical centre */ align-items: baseline; /* align text baselines */ } /* The famous "centre everything" combo: */ .container { display: flex; justify-content: center; align-items: center; }

Lesson 04 of 05
flex-grow & flex-shrink
Control how items expand to fill space or shrink when space is tight.
The flex shorthand

Items can respond to available space. flex-grow lets an item absorb extra space. flex-shrink lets it give up space when the container is too small. The shorthand flex combines both, plus flex-basis (the starting size).

CSS
/* flex: grow shrink basis */ .item { flex: 0 1 auto; } /* default */ .item-grow { flex: 1 1 auto; } /* take all spare space */ .item-double { flex: 2 1 auto; } /* take 2x more space */ .item-fixed { flex: 0 0 200px; } /* never grow or shrink */ /* Common shorthand values */ .item { flex: auto; } /* grow + shrink freely */ .item { flex: none; } /* rigid — never flex */ .item { flex: 1; } /* flex-grow: 1, shorthand */

flex-basis
The item's starting size before flex adjustments. Like width, but for flex.
flex: 1 shorthand

Lesson 05 of 05
flex-wrap
Allow items to spill onto new lines instead of shrinking to fit — the key to responsive layouts.
Why wrapping matters

By default, flex items never wrap — they squeeze onto one line no matter how small they get. Set flex-wrap: wrap and items that don't fit will naturally flow to the next row, giving you a responsive grid with no media queries.

CSS
.container { display: flex; flex-wrap: nowrap; /* default — single line */ flex-wrap: wrap; /* wrap to next row */ flex-wrap: wrap-reverse; /* wrap upward */ } /* Responsive card grid — no media queries needed! */ .card-grid { display: flex; flex-wrap: wrap; gap: 16px; } .card { flex: 1 1 200px; /* grow, shrink, min 200px wide */ }
