HTML Tags
Tags are the building blocks of every webpage. They tell the browser what kind of content to display — a heading, a paragraph, an image, and more.
What is a tag?

An HTML tag is a keyword wrapped in angle brackets: <tagname>. Most tags come in pairs — an opening tag and a closing tag with a slash. Everything between them is the content.

HTML
<h1>My Name</h1> ← heading <p>I love building things.</p> ← paragraph <img src="photo.jpg"> ← image (no closing tag) <div> <p>I can wrap other tags.</p> </div> ← container
Tags you'll use for a profile card
Live Reference

<h1> – <h3> → Your name, title Headings
<p> → A paragraph of text — your bio note Text

💡 Tags don't make things look good — they just give the browser meaning. CSS (which you'll learn in lessons 3 & 4) is what makes them look great.

Self-closing

🗂️

HTML Images
The <img> tag lets you display a photo, avatar, icon, or any picture on your page.
The img tag

Unlike most tags, <img> is self-closing — it has no content between tags. Instead, you describe the image through attributes. The two essential ones are src (where the image lives) and alt (a text description).

HTML
/* Basic */ <img src="photo.jpg" alt="My photo"> /* With size control */ <img src="photo.jpg" alt="My photo" width="80" height="80"> /* For a profile card: circular with CSS */ <img src="avatar.jpg" alt="Arjun K" class="avatar">

CSS Padding
Padding is the space inside an element — between its content and its border. It makes things breathe.
How padding works

Think of padding as the cushion inside a box. If a profile card has no padding, the text is jammed against the edges. Padding pushes the content inward, away from the walls of the element.

CSS
.card { padding: 24px; /* all 4 sides */ } .card { padding: 16px 24px; /* top/bottom left/right */ } .card { padding-top: 20px; /* individual sides */ padding-right: 24px; padding-bottom: 20px; padding-left: 24px; }

Inside the box

🎨

📐
Units
Use px for fixed, rem for scalable, or % for relative.
✍️
Shorthand
padding: 16px 24px = top-bottom 16, left-right 24.

Lesson 04 of 04
CSS Margin
Margin is the space outside an element — the breathing room between it and its neighbours.
Padding vs. Margin

If padding is the cushion inside the box, margin is the empty space around the outside of the box. It pushes other elements away. For a profile card, margin controls how much space separates the card from the rest of the page.

CSS
.card { margin: 32px; /* all 4 sides */ } .card { margin: 0 auto; /* centre horizontally */ } .name { margin-bottom: 8px; /* space below name */ } .bio { margin-top: 12px; /* space above bio */ }
